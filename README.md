# uclaradio-blog

### Start

- Open a new terminal.
- Compile the jsx files: `npm run compile`
- Run the program: `npm start`
- Navigation
  - Home -> [localhost:3000/]()
  - Login -> [localhost:3000/keystone]()

### View Database

- Navigate to [localhost:3000/keystone]()
- Login with user credentials in `updates/0.0.1-admin.js`
